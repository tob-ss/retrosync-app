# RetroSync

RetroSync is a cloud saving software for retro games! Backup your retro game saves into the cloud and sync them across multiple devices.

You can download RetroSync using the instructions down below!

## Installation

To install RetroSync click on the releases tab and download the latest release!

## Usage

Once you've installed RetroSync, you can access the dashboard via the RetroSync website. The dashboard allows you to interact with any device you've installed RetroSync on. 

You can also click on the RetroSync icon in the app tray to interact with it there.

## Contributing

RetroSync is closed source for now, however, plans are in place to make it open source. Especially for those who wish to use it in a home lab environment.
